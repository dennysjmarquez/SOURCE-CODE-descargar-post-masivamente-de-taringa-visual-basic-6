DESCRIPCIÓN

Nuevo Windows 7 ATI Edition 2010, disponible para procesadores de 32 BIT y 64 BIT. Completamente desatendido y basado en la versión Ultimate del útlimo sistema operativo de Microsoft. Al igual que Windows 7 NVIDIA Edition 2010, también creado por Benjamin del Rockers International Team. La imagen ISO original de Windows 7 Ultimate está en inglés, pero mediante Windows Update puede conseguirse el MUI en español para traducirlo a nuestro idioma.

REQUISITOS

· 1 GB de RAM (se recomienda 2 GB de RAM para 64-Bit)
· 20 GB de espacio en disco duro
· 128 MB de memoria de gráficos (Capacidad para ejecutar Aero)
· 17 “pulgadas del monitor (1024 x 76
· DVD-ROM o DVD-RAM
· Pixel Shader 2.0
· Dual Core de 64 bits OS Capaz
. La conexión a Internet activa (para la activación y actualizaciones y cambio de idioma)
